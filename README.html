Website with GPS Coordinates and Address Conversion

Features

  • Button 1: Retrieves the GPS coordinates of the device and displays them in the first text area.
  • Button 2: Converts the GPS coordinates into an actual address and shows it in the second text area.
  • Button 3: Validates a user-entered address against the converted address.
    • If the addresses match, the button turns green.
    • If the addresses do not match, the button turns red.

How to Use

  1. Open the website in a web browser.
  2. Ensure that your device has location services enabled.
  3. Click Button 1 to retrieve the GPS coordinates of your device.
  4. Click Button 2 to convert the GPS coordinates into an actual address.
  5. Enter an address in the third text area that you think matches the converted address.
  6. Click Button 3 to validate the entered address against the converted address.
  7. If the addresses match, the button will turn green. If they don't match, the button will turn red.
